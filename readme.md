# Wedding Planning System (CLI + SQLAlchemy ORM)

A command-line based wedding planning system built with Python, SQLAlchemy, and a clean modular structure.

## Features
- User management (Admin, Planner, Bride, Groom, Guest)
- Guest list management
- Vendor management
- Tasks & scheduling
- Budget tracking
- Event planning & seating assignment

## Technologies
- Python 3.12
- SQLAlchemy
- Pipenv Virtual Environment
